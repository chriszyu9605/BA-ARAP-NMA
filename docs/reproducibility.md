# Reproducibility

The standard installation uses the dependency ranges declared in `pyproject.toml` and supports Python 3.10–3.13.

`requirements-lock.txt` and `environment.yml` record the exact environment used for release validation: Python 3.13.5, NumPy 2.3.5, and SciPy 1.17.0. The lock file is not intended as a cross-version lock for Python 3.10–3.12.

The repository contains the complete source code, examples, tests, validation records, and user documentation needed to install and use the software.
